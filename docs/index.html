Features

Scatter Plot

Interactive readRatio vs zStrains scatter plot for any gene. Color-coded by interaction strength. Hover for details, click to inspect pairs.

GI Profile Correlation

Real-time Pearson correlation of any gene's GI profile against all ~1,500 genes. Instantly find functionally related genes and highlight them on the plot.

Gene Detail & SVG Export

View all interaction partners for any gene. Inspect pair-level metrics. Download scatter plots as publication-ready SVG files.

Quick Start

1

Clone & install

Requires Node.js v18+

2

Download data

Get small.zip from Figshare and extract to ../dual_tnseq_data/small/

3

Run

Data auto-imports into SQLite on first launch (~30 sec). Then search any gene.

# Clone
git clone https://github.com/shamlokto/dual-tnseq-explorer.git
cd dual-tnseq-explorer
npm install

# Download data from Figshare, extract to ../dual_tnseq_data/small/

# Start
npm run dev
